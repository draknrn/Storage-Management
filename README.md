# AWS Labs Portfolio 

##  Descripción general
Este repositorio contiene una colección de **laboratorios prácticos de Amazon Web Services (AWS)** que he desarrollado y documentado como parte de mi proceso de aprendizaje y formación en **Cloud Computing**.

El objetivo principal de este repositorio es **demostrar experiencia práctica real**, buenas prácticas y comprensión de servicios AWS, incluso sin contar aún con experiencia laboral formal en el área.

Cada laboratorio está **documentado paso a paso**, incluye **evidencia visual**, comandos reales y una sección de **mejoras para entornos de producción**, simulando escenarios que se presentan en el mundo profesional.

---

##  Objetivos del repositorio
- Consolidar conocimientos prácticos en AWS
- Documentar laboratorios de forma clara y profesional
- Demostrar habilidades técnicas a reclutadores y equipos técnicos
- Practicar buenas prácticas de seguridad, automatización y arquitectura cloud
- Construir un portafolio técnico verificable en GitHub

---

##  ¿Qué encontrarás en este repositorio?
Cada laboratorio incluye:

- Documentación en **Markdown**
- Descripción de la arquitectura utilizada
- Pasos de implementación con AWS Console y AWS CLI
- Validaciones y pruebas reales
- Capturas de pantalla como evidencia
- Enfoque en seguridad y principio de mínimo privilegio
- Sección: **“Qué haría distinto en producción”**

---

##  Servicios de AWS cubiertos
Este repositorio incluye laboratorios relacionados con:

- Amazon EC2
- Amazon S3
- Amazon EBS
- AWS IAM
- Amazon SNS
- AWS CLI
- Automatización con Python y Cron
- (y más a medida que el repositorio crece)
